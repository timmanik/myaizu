# @aizu/shared

This package contains shared TypeScript types and constants used across the frontend and backend.

## Responsibilities

- Define API-facing types shared across packages
- Export common enums, roles, prompt metadata, and constants
- Keep frontend and backend contracts aligned

## Scripts

```bash
pnpm dev
pnpm build
pnpm typecheck
pnpm clean
```

## Exports

- `@aizu/shared`
- `@aizu/shared/types`
- `@aizu/shared/constants`

Build output is published to `dist/` for workspace consumption.
