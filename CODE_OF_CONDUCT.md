# Code of Conduct

## Our Standard

We want the Aizu community to be welcoming, respectful, and constructive.

Examples of behavior that contribute to a positive environment include:

- Being respectful in disagreement
- Giving and receiving feedback gracefully
- Assuming good intent while discussing technical tradeoffs
- Focusing on what is best for the project and community

Examples of unacceptable behavior include:

- Harassment, insults, or personal attacks
- Discriminatory or exclusionary language
- Trolling, intimidation, or sustained disruption
- Publishing private information without permission

## Scope

This code of conduct applies in project spaces and in public interactions representing the project.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the project maintainer privately rather than opening a public issue.
